# 價格系統 v34

本版已包含：
- 查價前台：預設市場報價，並可用「＋」展開成本、利潤率、各經銷價格。
- 建立報價單：10 列下拉選單選商品、商品圖片、規劃費/設定費倍數、派工選項、即時計算利潤率/稅額/總價/收款節點。

# 智崴物聯價格系統 v33

這一版已經調整成較接近你目前的使用方式：

- 查價前台預設顯示市場報價
- 市場報價旁有 `+` 可展開更多欄位
- 可切換是否顯示成本
- 同頁顯示利潤率
- 報價單建立頁改成 **10 列下拉選單選商品**
- 每列可看到商品圖片、型號、品名、價格
- 可設定 **規劃費倍數 / 設定費倍數**
- 可選擇派工費（基本 / 半天 / 一天）
- 建立前先看到：
  - 產品合計
  - 規劃費
  - 設定費
  - 派工費
  - 未稅小計
  - 稅額
  - 含稅合計
  - 利潤率
  - 15 萬以上 / 以下的收款節點金額
- 商品備註會自動帶入報價單備註，可手動改
- 報價單頁面、Excel、PDF 都會帶出圖片欄位（有圖片時）
- 商品可手動新增與修改，也可自行上傳圖片

## 啟動方式

1. 完整解壓縮整個資料夾
2. 把 `202507新式報價單.xlsm` 放在與 `run.bat` 同一層（若你要重新匯入）
3. 雙擊 `run.bat`
4. 打開瀏覽器：
   `http://127.0.0.1:8000`

## 若要重新匯入 Excel

進入系統後打開：
- `後台匯入`

可重新將 Excel 內容匯入到系統。

## 匯率 / 折扣建議

建議保留現在這種做法：

1. 先以市場報價為基準
2. 再用匯率 / 折扣設定去推算：
   - 分公司
   - 總經銷商
   - 一級經銷商
   - 二級經銷商
   - 設計師
   - 市場最低價

這樣後續調整比較穩，也不需要每個商品都重改一次。
